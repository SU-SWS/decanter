Decanter Color System

Color (function)

color(color, $palette) will fetch a named color from the specified color map. If a color map is not specified, it will fetch the color from the default color map.

Example:

div {
  background-color: color(blue);
  color: color(reverse-text);
}

Color theme (mixin)

As you can see, calling a different palette when you have a large block of code can get rather repetitive. This is where the color theme mixin comes into play.

color-theme will temporarily override the default color map with the attributes of the specified $palette. This change is scoped to within the mixin block, and once the mixin has ended, the default color map will revert to its original state as defined by $color

Example:

$custom-color-palette: (background: #003111, text: #F4BFF0);
@include color-theme($custom-color-palette, '.example-3') {
  background-color: color(background);
  color: color(text);
}

Color themes (mixin)

color-themes consumes a map to render multiple color variations on a specific block. For each theme group in the theme group map, the $color-themes mixin iterates over that group with the specified selector.

Example:

SCSS:
$theme--blue: (background: #192f8c, text: #8cccf4);
$theme--green: (background: #003111, text: #b5f4c8);
$theme--orange: (background: #ad6200, text: #f4ce97);
$custom-theme-group: (
    default:  ( palette: $colors, selector: 'p' ),
    blue: ( palette: $theme--blue, selector: 'p.theme--blue' ),
    green: ( palette: $theme--green, selector: 'p.theme--green' ),
    orange: ( palette: $theme--orange, selector: 'p.theme--orange' ),
);

.example-4 {
  @include flex-items-per-row(4, $children: 'p');
  @include color-themes($custom-theme-group) {
    color: color(text);
    background-color: color(background);
    padding: 1em;
  }
}
